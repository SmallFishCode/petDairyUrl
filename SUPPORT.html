技术支持

易划 - 您的智能照片管理助手

欢迎使用易划！如果您在使用过程中遇到任何问题，请查看以下常见问题解答。如果问题仍未解决，请通过页面底部的联系方式联系我们，我们会尽快为您提供帮助。

1. 常见问题解答

Q1: 如何授权应用访问照片库？

首次打开应用时，系统会自动请求照片库访问权限。如果之前拒绝了权限，您可以：

  1. 打开 iOS 设置
  2. 找到"易划"应用
  3. 进入"照片"选项
  4. 选择"所有照片"或"选中的照片"
Q2: AI智能管家如何使用？

AI智能管家支持通过自然语言对话管理照片。使用方法：

  1. 在底部导航栏点击"小易AI"图标
  2. 在输入框中输入您的需求，例如：
    • "找水果照片" - 搜索包含水果的照片
    • "把所有水果照片加入到水果相册" - 创建相册并整理照片
    • "删除水果相册" - 删除指定相册
    • "删除第二张" - 删除重复照片中的第二张
    • "今天的照片" - 查看今天的照片
  3. 等待AI处理并执行相应操作
Q3: 如何删除照片？

删除照片有两种方式：

  1. 在照片浏览页面：向上滑动照片即可删除
  2. 通过AI智能管家：使用自然语言指令删除，例如"删除第二张照片"

删除的照片会进入垃圾箱，您可以随时恢复。点击右上角的垃圾箱图标可以查看已删除的照片。

Q4: 如何恢复误删的照片？

恢复照片的步骤：

  1. 点击右上角的垃圾箱图标
  2. 在垃圾箱中找到要恢复的照片
  3. 选择恢复或永久删除

您也可以使用撤销功能恢复最近删除的照片。

Q5: 如何将照片添加到相簿？

添加照片到相簿的方法：

  1. 在照片浏览页面，向下滑动到底部
  2. 可以看到相簿列表
  3. 点击相簿即可将当前照片添加到该相簿

您也可以通过AI智能管家批量整理照片到相簿。

Q6: 订阅会员后如何查看订阅状态？

查看订阅状态的方法：

  1. 打开 iOS 设置
  2. 点击您的 Apple ID
  3. 进入"订阅"选项
  4. 找到"易划"即可查看订阅详情

您也可以在应用的"更多"页面查看订阅状态。

Q7: 如何取消订阅？

取消订阅的步骤：

  1. 打开 iOS 设置
  2. 点击您的 Apple ID
  3. 进入"订阅"选项
  4. 找到"易划"并点击
  5. 选择"取消订阅"

取消后，您仍可在当前订阅周期内使用所有功能，到期后将无法继续使用高级功能。

Q8: 如何查看成就和清理统计？

查看成就的方法：

  1. 在底部导航栏点击"成就"图标
  2. 可以查看每天删除的照片统计
  3. 点击日历上的任意日期，可以查看当天的详细清理数据
  4. 颜色越深表示当天清理的照片越多
Q9: 照片分析功能需要多长时间？

照片分析的时间取决于照片数量和设备性能：

  • 首次使用应用时，系统会在后台自动分析照片
  • 分析过程可能需要一些时间，请耐心等待
  • 分析完成后，您可以使用AI搜索和整理功能
  • 新添加的照片会自动进行分析
Q10: 应用支持哪些类型的媒体文件？

易划支持以下类型的媒体文件：

  • 照片（JPG、PNG、HEIC等）
  • 实况照片（Live Photos）
  • 视频（MP4、MOV等）

2. 功能使用指南

2.1 AI智能管家

AI智能管家是易划的核心功能，支持通过自然语言对话管理照片。常用指令示例：

  • 搜索照片："找水果照片"、"显示今天的照片"、"找包含文字的照片"
  • 整理照片："把所有水果照片加入到水果相册"、"把今天的照片整理到今日相册"
  • 删除照片："删除第二张"、"删除所有重复照片"、"删除水果相册"
  • 查看信息："今天删除了多少照片"、"显示照片信息"

2.2 照片浏览

在首页可以按时间查看照片：

  • 最近照片：查看最近添加的照片
  • 按天浏览：按日期分组查看照片
  • 按月浏览：按月份分组查看照片

2.3 相簿管理

相簿功能帮助您更好地组织照片：

  • 在"相簿"页面可以查看所有相簿
  • 点击相簿可以查看相簿内的照片
  • 在照片浏览页面可以添加照片到相簿
  • 通过AI智能管家可以批量整理照片到相簿

2.4 成就系统

成就系统记录您的清理习惯：

  • 查看每天删除的照片数量和大小
  • 通过热力图查看清理习惯和强度
  • 颜色越深表示当天清理的照片越多
  • 点击日期可以查看当天的详细清理数据

3. 故障排除

3.1 应用无法访问照片

  1. 检查是否已授予照片库访问权限
  2. 前往 iOS 设置 > 易划 > 照片，确认权限设置
  3. 如果权限已授予但仍无法访问，尝试重启应用
  4. 如果问题仍然存在，请重启设备

3.2 AI功能无法使用

  1. 确认您已登录账号（使用Apple登录）
  2. 检查网络连接是否正常
  3. 确认订阅状态是否有效（部分功能需要会员）
  4. 尝试重新登录账号

3.3 照片分析速度慢

  1. 照片分析需要一定时间，请耐心等待
  2. 确保设备有足够的存储空间
  3. 关闭其他占用资源的应用
  4. 保持应用在后台运行，让分析继续进行

3.4 订阅状态异常

  1. 检查网络连接
  2. 前往 iOS 设置 > Apple ID > 订阅，确认订阅状态
  3. 尝试在应用内重新验证订阅
  4. 如果问题仍然存在，请联系我们

3.5 应用崩溃或卡顿

  1. 关闭应用并重新打开
  2. 重启设备
  3. 检查设备存储空间是否充足
  4. 更新应用到最新版本
  5. 如果问题持续存在，请通过反馈功能联系我们

4. 系统要求

易划对设备的要求：

  • iOS版本：iOS 15.0 或更高版本
  • 设备：iPhone、iPad
  • 存储空间：建议至少保留 500MB 可用空间
  • 网络：部分功能需要网络连接（AI功能、订阅验证等）

5. 隐私和安全

易划非常重视您的隐私保护：

  • 所有照片分析都在本地设备上进行
  • 照片内容不会上传到服务器
  • 只有照片的元数据和分析结果会存储在本地
  • 账号信息通过Apple安全系统管理

更多详细信息，请查看我们的隐私政策。
