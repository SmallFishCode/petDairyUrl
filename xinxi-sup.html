2. 常见问题

2.1 短信过滤不生效

如果短信过滤功能无法正常工作，请按以下步骤检查：

  1. 打开 iPhone 设置 → 信息 → 短信过滤
  2. 确保 信析 的开关已开启
  3. 在信析应用中，确保 AI智能过滤 开关已开启
  4. 重启 iPhone 后再次测试

2.2 AI分类不准确

AI分类结果可能不准确的原因：

  • 短信内容较为特殊，AI模型暂时无法准确识别
  • 模型需要更多训练数据来提升准确性
  • 您可以使用 投喂AI 功能帮助改进模型

您也可以使用 关键词拦截 功能手动添加需要拦截的关键词。

2.3 订阅相关问题

如何恢复购买？

  1. 打开信析应用
  2. 进入 账户 → 订阅管理
  3. 点击 恢复购买
  4. 按照提示完成恢复流程

订阅未生效？

  • 检查 App Store 中的订阅状态
  • 尝试退出并重新登录 Apple ID
  • 重启应用
  • 如果问题仍然存在，请联系我们

2.4 应用崩溃或闪退

如果应用频繁崩溃或闪退：

  1. 确保您的 iPhone 系统版本为 iOS 16.0 或更高版本
  2. 尝试重启 iPhone
  3. 卸载并重新安装应用
  4. 如果问题持续，请通过邮件联系我们，并提供以下信息：
    • iPhone 型号和系统版本
    • 应用版本号
    • 崩溃发生的具体操作步骤

2.5 数据同步问题

信析采用本地优先的设计：

  • 所有短信分类和过滤均在您的设备本地完成
  • 关键词和规则设置仅存储在您的设备上
  • 如果您更换设备，需要重新配置规则

3. 功能说明

3.1 AI智能过滤

AI智能过滤功能使用机器学习模型自动识别和分类短信：

  • 所有处理在您的设备本地完成，不会上传短信内容
  • 支持识别垃圾短信、验证码、通知等多种类型
  • 您可以通过 投喂AI 功能帮助改进模型准确性

3.2 关键词拦截

关键词拦截功能允许您手动添加需要拦截的关键词：

  • 支持添加多个关键词
  • 可以随时启用或禁用特定关键词
  • 关键词匹配不区分大小写

3.3 投喂AI

投喂AI功能帮助改进AI模型的准确性：

  • 粘贴短信内容并选择正确的分类
  • 系统会自动脱敏处理敏感信息（手机号、验证码等）
  • 仅发送脱敏后的文本用于模型训练

4. 系统要求

  • iOS版本：iOS 16.0 或更高版本
  • 设备：iPhone、iPad
  • 存储空间：建议至少 50MB 可用空间

5. 隐私和安全

重要提示：

  • 信析不会收集、存储或上传您的短信内容
  • 所有短信处理均在您的设备本地完成
  • AI模型在您的设备上运行，不会将数据发送到服务器
  • 我们严格遵守隐私保护原则，详见 隐私政策
